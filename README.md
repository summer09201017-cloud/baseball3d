# ⚾ 3D 棒球・主審視角(Baseball 3D)

Vite + Three.js 的 3D 棒球——鏡頭在捕手後上方(主審視角),球從投手丘由遠而近朝你飛來。
參考 `籃球CO`(fullcourt-3d-basketball)架構;規則沿用 2D「棒球打擊王」(九宮格好壞球、五種球種、時機窗)。

## 玩法

- **🤖 打擊練習**:阿福投 10 球你打,看總分拿星。
- **🎯 投球挑戰**:你投球(九宮格選格+五球種),阿福打擊。
- **⚾ 短場對戰**:3 局攻防——上半你打、下半你投,得分多的贏。
- **👥 雙人同機**:投打輪流 PK。
- **盜壘**:打擊方按 `E`(或 🏃 盜壘鈕)指揮前位跑者盜壘,失敗=出局。
- 雙鏡位:投打=主審近景;打出去/盜壘/跑壘=自動拉高看全場(四壘包、內外野守備、全壘打牆、觀眾席)。

## 操作

| 情境 | 鍵盤 | 觸控 |
|---|---|---|
| 打擊 | 空白鍵 / Enter 揮棒 | 點畫面或「揮棒!」鈕 |
| 盜壘 | E | 🏃 盜壘鈕 |
| 投球 | W/S 高低、A/D 左右(5×5 九宮格+外圈壞球)、Q/E 換球種、空白鍵投 | 「投球!」鈕(預設瞄準格) |

## 難度五檔

幼兒(超簡單)/兒童(簡單)/入門/標準/職業——時機窗寬度、球速、AI 投手壞球率與球種數、AI 打者實力全套調整。

## 中文播報(人聲)

播報字幕條+**預烤 mp3 人聲**(微軟雲哲神經語音;`node scripts/gen-voice.mjs` 烤製,產物進 git 離線可用)。
★人聲鐵則:不用 Web Speech 機器聲——沒烤過的句子只出字幕不唸。
新增播報詞:改 `src/voicePhrases.js` 再重跑烤製。

## 開發

```bash
npm install
npm run dev      # 開發
npm run build    # 建置 dist/
node scripts/gen-voice.mjs  # 烤播報人聲(需網路)
```
